Objetivo desse site

Neste trabalho, criei um site preliminar para a disciplina SI401, abordando todos os aspectos relevantes, como ementa, objetivos, tópicos, referências e critérios de avaliação. Utilizei HTML semântico para estruturar o layout e adicionei formatação com CSS externo. Garanti que todos os documentos gerados fossem compatíveis com HTML5 e validados nos sites do W3C, assegurando a acessibilidade e a funcionalidade do site. Foi uma oportunidade valiosa para aplicar meus conhecimentos em desenvolvimento web e seguir as melhores práticas.
